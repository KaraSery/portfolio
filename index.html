VRP Solver

VRP Solver est une application permettant d'optimiser des tournée de livraison entièrement codé en JavaScript coté client. Ce projet utilise les Web Components pour une interface modulaire, Redux pour la gestion de l’état ainsi qu'un algorithme permettant d'optimiser les trajets, offrant une solution performante et interactive directement depuis le navigateur.
